# configs/ - 配置檔案

此目錄包含機器人的配置檔案，支持標準化架構配置。

## 文件說明

- `settings.json` - 主要配置檔案，包含路徑配置、機器人設定、日誌設定等

## 配置項目

### 基礎配置
- 路徑配置
- 功能開關
- 日誌設定
- 機器人參數

### 標準化架構配置
- **命令權限配置**：支持 `public`、`basic`、`authorized`、`admin` 權限級別
- **跨頻道交互配置**：群組回覆開關、交互超時設置
- **安全性配置**：白名單管理、密碼驗證設置

### 功能配置
- **QR 登入**：啟用/停用
- **持久化會話**：啟用/停用  
- **群組回覆**：啟用/停用
- **訊息記錄**：啟用/停用
- **自動下載媒體**：啟用/停用
